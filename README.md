
# Project UAS Algoritma Pemrograman

Project yang dibuat untuk memenuhi tugas project pengganti UAS yang didalam nya mengandung Binary Search Tree dan Algoritma Djikstra. Di design untuk mudah di pahami dan interaktif sehingga dapat mendukung proses pembelajaran.


## Visualisasi Binary Search Tree (BST) & Algoritma Dijkstra

Project ini adalah aplikasi web edukasi interaktif untuk mempelajari dan memvisualisasikan:
- Binary Search Tree (BST): insert, search, reset, log langkah-langkah, dan visualisasi tree.
- Algoritma Dijkstra: input graf berbobot, jalankan Dijkstra dari node awal, visualisasi step-by-step, dan hasil jarak terpendek.
## Tech Stack

**Client:** React, TailwindCSS

**Server:** Vercel

## Run Locally

Clone the project

```bash
  git clone https://github.com/RedMagic46/Project-UAS-Algoritma-Pemrograman.git
```

Go to the project directory

```bash
  cd Project-UAS-Algoritma-Pemrograman
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Build Production

```bash
  npm run dev
```

Output build ada di folder ```build/```.
